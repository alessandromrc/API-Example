# API-Example
Api based on NodeJs with some endpoints including AI scannner image upload 

# Getting Started

### Prerequisites

1. NodeJS
2. MongoDB

## **Installing**

Run ```npm install``` to install all the dependencies

## **Start The API**

Run ```npm start``` to start the API
